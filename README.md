
# Vite + React Static Website


This is a Vite + React static website prepared for universal static hosting (GitHub Pages, Netlify, Cloudflare Pages, and more).

## Local Development

To run the project locally:

```sh
npm install
npm run dev
```

The site will be available at http://localhost:8080 (or the port shown in your terminal).

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

This project is ready for static hosting. Build with:

```sh
npm run build
```

Deploy the contents of the `dist` folder to your preferred static host.
